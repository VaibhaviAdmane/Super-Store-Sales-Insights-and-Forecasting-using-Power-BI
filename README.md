 📊 Super Store Sales Dashboard — Power BI
 
 An interactive Power BI dashboard for analyzing retail sales performance, profit trends, and delivery insights across regions, categories, and customer segments.

 📌 Project Overview

This dashboard provides a comprehensive view of a Super Store's sales operations — enabling stakeholders to monitor KPIs, identify top-performing categories, understand customer behavior, and forecast future sales trends using real-time visual analytics.

📁 Dashboard Pages
🗂️ Page 1 — Sales Overview
A full-view operational dashboard covering all key sales metrics:

| Visual | Description |
| 📦 KPI Cards | Total Sales, Profit, Orders & Avg. Delivery Days |
| 📊 Bar Chart | Sales by Category & Sub-Category |
| 🚚 Bar Chart | Sales by Ship Mode |
| 🍩 Donut Chart | Sales by Segment, Payment Mode & Region |
| 🗺️ Map | Sales & Profit distribution by State |
| 📈 Area Chart | Sales & Profit trend by Month and Year |
| 🔘 Slicer | Region filter (East, West, Central, South) |

 🗂️ Page 2 — Sales Forecasting
- 📉 Time-series **15-day Sales Forecast** using Order Date trends
- 📊 State-wise sales comparison for regional performance tracking


 🎯 Key Metrics Tracked

- Total Sales — Overall revenue generated
- Total Profit — Net profit across all orders
- Total Orders — Volume of transactions
- Avg. Delivery Days — Operational efficiency indicator


🔍 Key Insights Enabled

- Identify which **categories and sub-categories** drive the most revenue
- Compare **shipping modes** to optimize logistics costs
- Understand **customer segment** and **payment mode** preferences
- Track **monthly and yearly** sales & profit trends
- Visualize **state-wise performance** across the US map
- Forecast **short-term sales** for inventory and capacity planning
